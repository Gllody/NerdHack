<h1 align="center">NerdHack</h1>
<p align="center">A Minecraft Fabric Utility Mod for anarchy servers. Based on Meteor Client.</p>

## Usage

### Building
- Clone this repository
- Run `./gradlew build`

### Installation
- Put this mod and Baritone mod in mods folder (.minecraft)
